# UI Flow Checklist

1. Start a new game.
2. Open the price menu.
3. Change a price.
4. Confirm the menu stays open.
5. Confirm only the selected product changed.
6. Confirm the price/menu button changes to the completed/changed color.
7. Open the orders menu.
8. Complete an order action.
9. Confirm the menu stays open.
10. Confirm the orders button changes to the completed/changed color.
11. Open staff menu.
12. Complete a staff action.
13. Confirm the menu stays open.
14. Confirm staff button changes to the completed/changed color.
15. Check that the old checklist is gone.
16. Try opening the store before tasks are complete.
17. Confirm no warning popup appears.
18. Confirm the small task box lists missing tasks.
19. Complete the listed tasks.
20. Confirm the task box updates.
21. Confirm store stats appear on the hotbar.
22. Confirm old above-store stats are gone.
23. Confirm the store display is smaller but still usable.
24. Confirm UI uses a coherent pixel-like warm palette.
25. Confirm dropdowns/buttons/panels/inputs share the same theme.
26. Confirm no overlapping text or UI elements.
27. Confirm Back/Escape closes the active menu.
28. Confirm no duplicate action execution occurs.
