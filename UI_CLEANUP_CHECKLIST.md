# UI Cleanup Checklist

Manual checks for the current UI/UX cleanup pass:

1. Start menu buttons are easy to click.
2. Start menu looks cleaner and more balanced.
3. Menus are less wide but still readable.
4. Romanian text does not clip.
5. Dropdowns are readable.
6. Price/orders/staff/reports menus fit comfortably.
7. Checklist is gone.
8. No empty checklist container remains.
9. No invisible checklist element blocks clicks.
10. Small task text box is readable.
11. Color scheme is calmer and warmer.
12. No neon or eye-straining colors remain.
13. Buttons, dropdowns, panels, and inputs share one coherent palette.
14. Store display has enough room.
15. Hotbar remains readable.
16. Back/Escape/Close still works.
17. No gameplay logic was accidentally changed.
